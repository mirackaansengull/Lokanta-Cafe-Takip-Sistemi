1)Veritabanı için  menu.bacpac dosyasını MSSQL de ekleyiniz.
2)Projeyi çalıştırmak için XAMPP in ve sqlsrv eklentilerinin bilgisayarınızda kurulu olması lazım.
3)Kurulumları tamamladıktan sonra  dosya klasörünü xammp dosya konumunun C:\xampp\htdocs\  benim için C diskinde olduğu için yolu böyle htdocs klasörünün içine atıyoruz.
4)Gerekli adımları uyguladıktan sonra xammpde apache serverını çalıştırdıktan sonra tarayıcıya girerek proje klasörünün path yolunu aratıyoruz.
5)Artık giriş yapabiliriz giriş için 3 şecenek var.

Kullanıcı Adı: admin 
Şifre: admin                 --> bu yol ile admin olarak giriş yaparsınız.


Kullanıcı Adı: lokantaci
Şifre: lokantaci                 --> bu yol ile lokanta sahibi olarak giriş yaparsınız.


Kullanıcı Adı: musteri 
Şifre: musteri                 --> bu yol ile müşteri olarak giriş yaparsınız.
